# Async Weather & News Dashboard

This project demonstrates asynchronous programming in Node.js + TypeScript using three styles:

- **Callbacks**
- **Promises**
- **Async/Await**

## Features
- Fetch weather data from Open-Meteo API
- Fetch news headlines from DummyJSON Posts API
- Implementations using callbacks, promises, and async/await
- Demonstrates Promise.all() and Promise.race()

## Run Scripts
```bash
npm run callback
npm run promise
npm run async
```

## Learning Outcomes
- Understanding of event loop
- Error handling in async code
- Practical use of APIs
